# True Capture Website

This is your complete True Capture website package.

## File Structure

```
true-capture-website/
├── index.html                              # Homepage (What We Do)
├── archival-workflow.html                  # Our Archival Workflow page
├── archival-transfer-services.html         # Archival Transfer Services page
├── order-services.html                     # Place an Order page
├── styles.css                              # Main stylesheet
├── script.js                              # JavaScript functionality
├── *.svg                                  # SVG assets (logos, waveforms)
├── *.webp                                 # Image assets (tape formats)
└── assets/
    └── img/                               # Duplicate assets for different references
        ├── *.svg
        └── *.webp
```

## Pages Included

1. **index.html** - Homepage with format accordion, comparison section, and sticky waveform
2. **archival-workflow.html** - Workflow comparison and equipment details
3. **archival-transfer-services.html** - Service details with format descriptions
4. **order-services.html** - Order form and contact information

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Mobile navigation toggle
- ✅ Expandable format accordion on homepage
- ✅ Sticky waveform effect on homepage
- ✅ Professional workflow comparison
- ✅ All tape format images included (VHS, Hi8, MiniDV, VHS-C)
- ✅ Complete branding assets (wordmarks, logos)

## How to Use

1. Upload all files to your web server
2. Make sure the directory structure is preserved
3. The site should work immediately (all assets are relative paths)

## Browser Compatibility

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile responsive
- Accessibility features included

## Notes

- Some HTML files reference images in the root directory
- Some HTML files reference images in assets/img/
- Both locations are included for compatibility
- No build process required - pure HTML/CSS/JS
