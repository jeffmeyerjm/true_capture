File formats explained

FFV1 Raw Interlaced

Mathematically lossless capture preserving the original interlaced scan structure. This is the digital negative - the closest thing to the original tape. Available on request for advanced archival needs.

ProRes MOV Mezzanine Master

Your working archival file. High-quality, edit-friendly, and preservation-grade. This is the file you'll use for editing, sharing, and long-term storage.

MP4 Viewing Copy

Optimized for playback on TVs, phones, tablets, and computers. Smaller file size for easy sharing while maintaining excellent quality.

File size guideline: A typical FFV1 raw capture file is 50-60 GB per 2-hour tape. MP4 viewing copies are significantly smaller, ~12 GB. Mezzanine working files are larger 60-110 GB (depending on options selected).
