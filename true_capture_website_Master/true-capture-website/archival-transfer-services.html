What you receive

Every archival transfer includes multiple file formats optimized for different uses.

  • ProRes MOV mezzanine master - High-quality, edit-friendly preservation file
  • MP4 viewing copy - Optimized for TVs, phones, and easy sharing
  • Optional raw FFV1 interlaced archive - Mathematically lossless capture preserving original scan structure
File sizes: A typical ProRes mezzanine master is approximately 50-60 GB per 2-hour tape. MP4 viewing copies are significantly smaller. FFV1 raw archives are larger and available on request.

Supported tape formats

We handle all common analog and DV tape formats with format-appropriate playback equipment.

VHS & S-VHS

Professional playback with line time-base correction. We use professional S-VHS decks with built-in line TBC to stabilize horizontal scanlines at playback. This reduces flagging, jitter, and timing wobble before any external processing occurs. S-VHS preserves higher luminance resolution when present on the source tape.

VHS-C

Compact VHS with proper adapters. VHS-C tapes are played using motorized adapters or direct VHS-C playback mechanisms. The same professional signal chain applies: line TBC correction, full-frame synchronization, and broadcast SDI capture to ensure stable, archival-quality transfers.

Video8 / Hi8

8mm analog camcorder formats. We use the Sony DCR-TRV330 camcorder with built-in line TBC for Video8 and Hi8 playback. Camcorder-recorded tapes perform best when played on equipment designed for them. Hi8 captures retain the extra luminance detail this format can provide when recorded properly.

Digital8

Digital video on 8mm tapes. Digital8 stores DV-compressed video on 8mm tape stock. We capture via FireWire/i.LINK to avoid unnecessary recompression, preserving the original DV stream. The Sony DCR-TRV330 handles Digital8 playback with proper tape transport and head alignment.

MiniDV

Digital tape with frame-accurate capture. MiniDV tapes store DV-compressed video digitally. We use FireWire/i.LINK capture to preserve the original DV stream without recompression. Frame-accurate timecode ensures every frame is captured correctly. The result is an archival-friendly file ready for long-term storage or editing.
